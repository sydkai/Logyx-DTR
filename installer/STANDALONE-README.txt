Logyx DTR — Standalone Offline Application
==========================================

This installation is fully self-contained on ONE computer.
No internet, LAN, or GitHub required.

WHAT IS INCLUDED
----------------
  - Logyx DTR attendance scanner and admin pages
  - Built-in SQLite database (logyx.db) with employees and admins
  - All runtimes bundled inside this folder

HOW TO START
------------
  Double-click: LogyxDTR.cmd
  Or use the Start Menu / Desktop shortcut: Logyx DTR

  The app opens in your default web browser (Chrome, Edge, etc.).
  The server runs quietly in the background — no extra window to keep open.

DATABASE LOCATION
-----------------
  Your live data is stored at:
  %APPDATA%\Logyx DTR\logyx.db

  Scans, leave requests, and employee changes are all saved here.
